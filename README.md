# 🏦 SecurePay NeoBank

A full-stack MERN-based digital wallet and neo-banking platform that combines secure financial transactions, KYC verification, fraud detection, and administrative monitoring in a single system.

## 🚀 Features

### 👤 User Features

* Secure Registration & Login
* OTP Verification & JWT Authentication
* Digital Wallet Management
* Add Money to Wallet
* Peer-to-Peer (P2P) Money Transfer
* Receive Money via Links & QR Codes
* Transaction History & Details
* Real-Time Notifications
* Security Settings & Profile Management

### 🛡️ KYC & Compliance

* Two-Level KYC Verification
* Document Upload & Validation
* Admin Review Workflow
* KYC Status Tracking
* Compliance-Oriented User Verification

### 🔍 Fraud Detection & Security

* Risk-Based Transaction Monitoring
* Fraud Alert Generation
* Suspicious Activity Detection
* Secure Authentication Middleware
* Session & Access Control
* Audit-Oriented Architecture

### 👨‍💼 Admin Panel

* User Management
* KYC Review & Approval
* Transaction Monitoring
* Fraud Alert Dashboard
* Administrative Controls & Analytics

## 🏗️ Tech Stack

### Frontend
